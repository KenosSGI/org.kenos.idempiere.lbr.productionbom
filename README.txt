							Plugin org.kenos.idempiere.lbr.production.bom
		
Este plugin alterar a Produção (Individual) para utilizar
a Lista de Material do IDempiere Libero (Tabelas PP_Product_BOM e PP_Product_BOMLine)
ao invés da Lista de Material padrão do IDempiere (Tabelas M_Product_BOM e M_Product_BOMLine)

Executar scripts para utilizar o recurso de Criar Lista de Material por Parceiro de Negócio e
para ocultar a Aba Lista de Material da janela Produto.
